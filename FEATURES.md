# Student Marketplace App: Feature List

This document outlines the planned features for the application, categorized by user impact area.

---

### Core User & Trust Features
-   **Verified Student Accounts:** Mandatory login with a `.edu` email address, which sends a verification link before the account is activated.
-   **User Profiles:** Shows username, profile picture, university, major (optional), graduation year, join date, and average rating.
-   **Seller/Buyer History:** Displays past items sold/bought and all reviews received.
-   **Ratings & Reviews:** 5-star rating system with public comments for both buyers and sellers after a transaction is marked complete.
-   **Report & Block System:** Easy-to-access buttons on listings and profiles to report inappropriate content/behavior or block users.

---

### Listing & Transaction Features
-   **Item Listings:** Include multiple photos/videos, a detailed description, category, subject code (if applicable), and mandatory condition tags (e.g., `New`, `Like New`, `Good`, `Fair`).
-   **Pricing Options:** Allow sellers to set a `Fixed Price`, mark it as `Negotiable`, or list it for `Free` or `Swap Only`.
-   **In-App Chat:** Real-time messaging with features like read receipts and quick-reply templates ("Is this still available?").
-   **Negotiation System:** A formal `Make Offer` button that sellers can `Accept`, `Reject`, or `Counter-offer`.
-   **Safe Meetup Suggestions:** When scheduling a meetup, the app can suggest pre-vetted, public, well-lit spots on campus (e.g., "Library Entrance," "Student Union Lobby").
-   **Mark as Sold:** A simple button for sellers to move a listing from active to sold, which prompts them to review the buyer.

---

### Search & Discovery Features
-   **Subject Name Search:** Search by subject name (e.g., "Calculus 101") to find all related textbooks, notes, and calculators.
-   **Advanced Filtering:** Filter search results by category, price range, condition, distance from a campus location, and "swap only" items.
-   **Wishlist:** Add a desired item (e.g., "Graphing Calculator TI-84") to a wishlist and get a notification if an item matching the description is listed.
-   **Categories:** Pre-defined categories like `Textbooks`, `Electronics`, `Dorm Furniture`, `Lab Equipment`, `Art Supplies`, etc.
